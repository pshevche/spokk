# Spockk IntelliJ plugin

Supports seamless integration of the Spockk testing and specification framework with IntelliJ.

![Screenshot](https://raw.githubusercontent.com/pshevche/spockk/refs/heads/main/spockk-intellij-plugin/docs/screenshot.png)

## Features
- Detect Spockk syntax in test sources
- Execute Spockk tests with Gradle from IDE
