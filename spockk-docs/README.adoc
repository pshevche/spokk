== spockk-docs

Contains the user guide content.
The documentation is written using https://asciidoctor.org/[Asciidoctor] and is transformed into HTML using the https://asciidoctor.github.io/asciidoctor-gradle-plugin/master/[Asciidoctor Gradle plugin].
The documentation gets deployed to https://pshevche.github.io/spockk[GitHub pages] using the "Publish" workflow.
