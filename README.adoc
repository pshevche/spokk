= Spockk Framework

This repository reimagines the https://github.com/spockframework/spock[Spock] testing framework for Kotlin.
The project is work-in-progress. See the approximate roadmap https://github.com/pshevche/spockk/milestones[here].

== Concept

The framework relies on a Kotlin compiler plugin to support an expressive syntax in your test code.

Example:

[source,kotlin]
----
fun `can add numbers`() {
    given("two numbers")
    val a  = 1
    val b = 2

    expect("correct sum")
    assert(a + b == 3)
}
----

== Modules

* `spockk-compiler-plugin`: contains the implementation of the IR transformer.
* `spockk-core`: testing engine and specification syntax;
* `spockk-gradle-plugin`: Gradle plugin that simplifies the application of the `spockk-compiler-plugin`.
* `spockk-specs`: specifications, implemented using Spockk.
