= spockk-core

Declares the specification syntax and implements the `TestEngine` capable of executing transformed tests.

== Noteworthy components

- link:src/main/kotlin/io/github/pshevche/spockk/lang/lang.kt[lang]: contains block labels that allow users to define feature methods. These blocks mark the features and statements that need to be transformed during compilations. They also help IntelliJ detect executable Spockk tests.
- link:src/main/kotlin/io/github/pshevche/spockk/runtime/SpockkTestEngine.kt[SpockkTestEngine]: JUnit Platform `TestEngine` that executes transformed specifications and features. It is heavily inspired by https://github.com/spockframework/spock/blob/master/spock-core/src/main/java/org/spockframework/runtime/SpockEngine.java[Spock's test engine].
